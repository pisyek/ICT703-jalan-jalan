# Jalan-Jalan

Smart Holiday Retreat Planning Platform - A cognitive travel planning website for ICT703 Human Centered Informatics.

## About

Jalan-Jalan helps travelers make informed, context-aware decisions by integrating:
- AI-powered planning assistance
- Interactive travel dashboards
- Personal travel analytics
- Community knowledge sharing
- Predictive insights

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
